# IRONPULSE — Performance Laboratory

A production-ready, dark industrial luxury gym marketing website built from Google Stitch design specs.

## 🔥 Live Preview
Open `index.html` in any browser — no build step required.

## 🗂️ Project Structure

```
├── index.html   # Full single-page website (semantic HTML5)
├── style.css    # Complete design system (Vanilla CSS)
├── app.js       # All interactivity (Vanilla JS)
└── README.md
```

## ✨ Features

- **Dark Industrial Luxury** design — Obsidian base + Electric Amber (#FFB800) accents
- Sticky navbar with transparent→glassmorphism scroll transition
- Hero section with parallax effect and animated scale
- About / Why Choose Us with animated stat counters
- Membership Plans — 3 tiers with Monthly/Yearly toggle, all CTAs → WhatsApp/Email
- Facilities masonry gallery with hover zoom
- Equipment showcase (4 zones)
- Trainers section with quote overlays & certification badges
- Testimonials auto-playing carousel with touch/swipe support
- Animated count-up stats (IntersectionObserver)
- Programs bento grid
- FAQ accordion
- Contact section — WhatsApp + Email only (no backend)
- Fully responsive (mobile → desktop)
- Scroll-reveal animations throughout

## 🎨 Design System

| Token | Value |
|---|---|
| Background | `#0A0A0A` |
| Surface | `#131313` |
| Primary Accent | `#FFB800` |
| Text | `#E5E2E1` |
| Headline Font | Oswald (700) |
| Body Font | Hanken Grotesk |
| Label Font | JetBrains Mono |
| Border Radius | 0px (sharp) |
| Transition | `cubic-bezier(0.19, 1, 0.22, 1)` |

## 🔧 Customisation

1. Replace `919999999999` with your WhatsApp number
2. Replace `info@ironpulse.in` with your email
3. Update gym name, prices, trainer bios, and photos
4. Swap Google Maps embed with your actual location

## 📋 Contact Integration

All "Join" / "Choose Plan" CTAs use:
- **WhatsApp** → `https://wa.me/<number>?text=<prefilled message>`
- **Email** → `mailto:<email>?subject=...&body=...`

No payment gateway, no backend, no database required.
